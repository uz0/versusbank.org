# Bounded Context

## Vocabulary
VersusBank: The banking/financial platform domain. Domain: The versusbank.org web platform. MCP: Model Context Protocol integration. Quint: Formal reasoning and verification system. Bounded Context: The scope and boundaries for reasoning about the VersusBank system.

## Invariants
Public domain licensing must be maintained. Domain name versusbank.org must be preserved. MCP server configuration must remain functional. Quint reasoning system must be properly initialized. Git version control must track all changes.
